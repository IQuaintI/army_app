How should an impalement injury be stabilized?

By placing several layers of bulky dressings around the injury site so that the dressing surrounds the object

When a protruding object is present in an abdominal wound, what should you do with it?

Stabilize it

If a casualty has an impalement injury in the cheek in which both ends of the object can be seen, how should it be removed?

Remove the object in the direction it entered
